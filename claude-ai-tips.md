# Claude.ai Tips

> Claude.aiの機能活用・Claude CodeやCoworkとの連携方法

## 現状（できていること）

### chart_display_v0（チャート表示機能）
- claude.aiの会話内にグラフを埋め込める専用ツール
- 対応チャート: 折れ線グラフ・棒グラフ・散布図
- データを渡すだけでシンプルに表示、スマホアプリでもOK
- インタラクティブ性はホバーで値が見える程度

### Reactアーティファクトとの使い分け

| | chart_display_v0 | Reactアーティファクト |
|---|---|---|
| 手軽さ | データだけでOK | コードを書く必要あり |
| 自由度 | 3種類のみ | 無限（アニメーション等も可） |
| 用途 | CSVやExcelの数値をパッと可視化 | バーチャートレース等リッチな表現 |

### 業務での活用例
- 竣工数量計算書のデータ可視化
- 工事の進捗や管材の数量比較
- 共通仕様書の検索結果件数の比較

### Claude CodeとClaude.aiの連携
- **Claude Codeはclaude.aiのチャットを直接参照できない**（別セッション）
- チャットの内容をClaude Codeに伝える方法:
  1. claude.aiで内容をMarkdownにまとめてファイル化 → Claude Codeに渡す（おすすめ）
  2. Claude Codeにテキストで要点を伝える（貼り付け）
  3. チャットURLを共有しても中身は読めない
- 引継書（HANDOVERファイル）を使うのが最も確実

### Claude.aiの役割（三層構成における位置づけ）
- 相談・学習・発想の場
- PCがなくてもスマホ・iPadで気軽に使える
- 新機能の体験、方針の整理、Markdownまとめ
- 過去チャット検索機能でナレッジの掘り起こし
